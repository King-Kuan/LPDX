# LightDoc Palace Studio

A polished, Android-friendly LightDoc-style document studio built with Next.js and TypeScript.

## Ownership
This project is created for **The Palace Tech House** and includes visible product-rights attribution in the interface and metadata.

## Highlights
- Mobile-first responsive layout
- Local-first document storage
- Rich block-based editor
- `.lpdx` export and import
- Quick template creation
- Snapshot stats and reading metrics
- Simple PDF export
- Android-friendly bottom navigation and sheets

## Run locally
```bash
npm install
npm run dev
```

## Build static output
```bash
npm run build
```
